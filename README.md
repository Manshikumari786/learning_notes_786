1. Sliding Window
   Sliding Window is a technique where we take a small “window” of continuous elements in an array or string and move it forward step by step, instead of recalculating everything again and again. It uses two pointers — start and end. The end pointer is used to expand the window (add new elements), and the start pointer is used to shrink the window (remove old elements) when a condition is not satisfied. This approach is mainly used in subarray or substring problems (like maximum sum or longest substring) and helps reduce time complexity from O(n²) to O(n).
